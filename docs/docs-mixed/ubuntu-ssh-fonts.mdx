# Fix SSH fonts on Ubuntu

If fonts look broken over SSH, install Meslo Nerd Font locally on the Ubuntu machine so your terminal can render symbols correctly.

```bash
sudo apt install fontconfig
cd ~
wget https://github.com/ryanoasis/nerd-fonts/releases/download/v2.1.0/Meslo.zip
mkdir -p .local/share/fonts
unzip Meslo.zip -d .local/share/fonts
cd .local/share/fonts
rm *Windows*
cd ~
rm Meslo.zip
fc-cache -fv
```
